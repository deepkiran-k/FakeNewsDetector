What is FactCheck?

FactCheck is the umbrella initiative for all fact-checkingâ€“related efforts within this project. It began as an attempt to build an AI system capable of automatically identifying check-worthy claims. Since then, it has steadily advanced toward the ultimate goal of fully automated fact-checking.

Who uses it?

FactCheck is mainly used by journalists, but in reality anyone interested in tackling misinformation can make use of it. Our API provides easy access to our models and is accessible by just registering for a free API key.

How can I contribute?

FactCheck is made possible by human data-labeling contributions. Feel free to sign up for an account and begin labeling to help us deliver better models. We have also open sourced our machine learning model training code, so if you are a savvy AI engineer feel free to make contributions there as well.
